# Sky-bank project
##  Описание:
Разрабатывается функционал банковского приложения.
## Установка:
1. Клонируйте репозиторий:
```
git clone https://github.com/ArtemMTim/DZ_project
```
2. Установите зависимости:
```
pip install -r requirements.txt
```
